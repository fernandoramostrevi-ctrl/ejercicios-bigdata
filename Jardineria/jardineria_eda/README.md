# Proyecto de Análisis de Datos de Jardinería

Este proyecto contiene un análisis exploratorio de datos (EDA) sobre datos de jardinería.

## Estructura del Proyecto

- `data/raw/`: Contiene los datos crudos generados.
- `data/processed/`: Contiene los datos procesados y limpios.
- `src/generate_data.py`: Script para generar datos sintéticos.
- `src/load_data.py`: Script para cargar y procesar los datos.
- `src/eda_notebooks/`: Contiene notebooks de Jupyter para el análisis.
- `src/app_dash.py`: Aplicación web con Dash para visualizar los datos.
- `requirements.txt`: Dependencias del proyecto.
- `README.md`: Este archivo.
